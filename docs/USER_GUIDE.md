# Guide Utilisateur - Metalyzr

Bienvenue sur Metalyzr ! Ce guide vous aidera à naviguer et à tirer le meilleur parti de notre plateforme d'analyse du métagame.

## 1. Accès à la Plateforme

Vous pouvez accéder au dashboard public de Metalyzr via votre navigateur web. Si vous utilisez un environnement de développement local, l'adresse est [http://localhost:3000](http://localhost:3000).

## 2. Interface Principale

L'interface est conçue pour être intuitive et vous donner un accès rapide aux informations les plus pertinentes. Elle se compose de plusieurs sections principales.

### a. Contrôles Globaux

En haut de la page, vous trouverez des filtres globaux qui affectent toutes les données affichées sur le dashboard :

- **Filtre de Format** : Choisissez le format de jeu qui vous intéresse (Standard, Modern, Pioneer, etc.).
- **Filtre de Période** : Sélectionnez la période d'analyse (7 derniers jours, 30 derniers jours, etc.).

### b. Navigation par Onglets

Le contenu principal est organisé en onglets pour une navigation facile :

- **Vue d'ensemble** : L'onglet par défaut, il offre un aperçu global du métagame.
- **Tournois** : Affiche une liste des tournois récents.
- **Explorer les Decks** : Permet de rechercher et filtrer des decklists spécifiques.

## 3. Fonctionnalités Détaillées

### a. Onglet "Vue d'ensemble"

- **Répartition du Métagame** : Un graphique (camembert) montre la part de chaque archétype majeur dans le métagame pour la période et le format sélectionnés.
- **Top Archétypes** : Une liste classée des archétypes les plus performants, avec des indicateurs de tendance (en hausse, en baisse, stable).

### b. Onglet "Tournois"

- Affiche une liste des tournois les plus récents, avec des informations clés comme le nom de l'événement, le nombre de joueurs et le deck du vainqueur.
- Cliquez sur un tournoi pour (prochainement) voir plus de détails.

### c. Onglet "Explorer les Decks"

- **Barre de Recherche** : Utilisez la barre de recherche pour trouver des decks en fonction du nom d'un archétype, d'un joueur ou d'une carte spécifique.
- **Filtres Avancés (Prochainement)** : Des options de filtrage plus fines seront disponibles pour affiner votre recherche.
- **Grille de Decks** : Les résultats s'affichent sous forme de cartes, chacune représentant une decklist. Cliquez sur une carte pour voir les détails complets du deck.

## 4. Export de Données

Sur certains graphiques et tableaux, un bouton **"Export"** vous permettra de télécharger les données brutes (au format CSV, par exemple) ou une image du graphique pour vos propres analyses ou créations de contenu.
